## 结论

当前主线不能简单写成“继续追算力”。更准确的排序是：

半导体 + AI硬件（中期五月核心仍在，但短线从追涨切到抗跌、回踩和修复确认） > 钠电 / 电池材料 / 氟化工（边际增强，偏脉冲和低吸） > 机器人 / 汽车零部件（异动增多，等待消息和板块合力） > 航天 / 商业航天 / 军工（次选但趋势受损，必须看修复） > 算力租赁 / 国产算力芯片（逻辑未灭，但多数算力进入调整，只留强票和支撑位反弹） > 电力、金属、化工分支轮动 > 创新药、防御线。

一句话说：名义主线还是 `半导体 + AI硬件`，但 5/15 的交易重心已经变成“科技高位不追、破位减仓；新机会看钠电、电池材料、氟化工和机器人脉冲；航天和算力租赁只做修复，不做弱票幻想”。

## 1. 半导体 + AI硬件：中期核心仍在，短线降追高

价值锚最新仍把主线写成 `半导体 + AI硬件`，并继续强调：

- AI硬件优先 `PCB / 电子布 / CCL / 铜箔`，电子布锚点包括中国巨石。
- 半导体材料只看少数核心，靶材锚点包括欧莱新材、江丰电子，磷化铟锚点包括云南锗业。
- 光通信只看少数核心，锚点包括光迅科技、东山精密、中天科技，但位置已经要小心。
- 先进封测前排、薄膜铌酸锂、石英股份仍在核心观察里。
- 当天能抗住板块下跌的品种，优先级高于单纯题材名字好听的票。

但主博 5/15 的变化很重要：英伟达芯片部分解封会压制国产算力芯片预期，寒武纪、摩尔线程这类国产算力承压；光模块和存储芯片相对受益，但兆易创新、江波龙、中际旭创等也需要技术调整，近期放量拉升不能追，最好等 `15-30%` 回踩再看。主博还直接提示“最近特别留意的不是算力了”，寒武纪、东山精密、长飞光纤等已经进入调整。

因此这一条主线现在要分两层处理：

- 中期产业主线仍在：PCB、电子布、CCL、铜箔、光通信核心、存储、先进封测、靶材、石英、薄膜铌酸锂。
- 短线交易节奏降权：不追高位算力和科技扩散，只看抗跌核心、缩量回踩、支撑位修复和反包确认。破趋势的票按纪律减仓，不因为板块名还在主线里就硬拿。

## 2. 钠电 / 电池材料 / 氟化工：边际增强，但仍按脉冲低吸

主博 5/15 明确把注意力从算力转到电池和钠电：

- 电池板块起来，多氟多领先。
- 钠电是关键，维科技术继续关注。
- 当前仍是脉冲行情，先“小吃”，回落后再来。
- 氟化工大涨偏电池方向，多氟多最强，巨化股份、金石资源跟随。
- 新和成、万华化学阶段见顶回落，后续氟化工重点更偏巨化股份。

这条线的定位是“短线边际增强”，不是无脑替代 AI硬件成为新总龙头。更适合的买法是回落低吸、分歧承接和龙头确认；如果当日 `watchlist` 没有直接对应的多氟多、维科技术、巨化股份、金石资源、湖南裕能、恩捷股份、天赐材料等方向，不要为了贴主线硬选弱相关股票。

## 3. 机器人：异动增多，等待板块合力和消息催化

主博 5/15 连续提到机器人：

- 领益智造表现不错，拓普集团、五洲新春同步，三花智控稍弱，板块合力开始出现。
- 柯力传感、北特科技、安培龙等异动，有全面爆发节奏。
- 三花智控、拓普集团仍是龙头节奏，五洲新春、万向钱潮弹性更大。
- 7-8 月特斯拉机器人量产预期提供支撑，但当前仍偏脉冲炒作，靠近前高要减，保留底仓。

机器人从此前的“技术反弹观察”上修为“边际增强的轮动主线”。但它还不是低风险单边主升，核心仍是回踩确认、消息兑现前后的节奏和板块合力。选股时可以提高机器人、智能执行器、汽车零部件弹性票的权重，但涨到前高附近要降仓位。

## 4. 航天 / 商业航天 / 军工：只做修复，趋势不修复就降级

价值锚最新对航天的态度变谨慎：商业航天整体趋势破了，若次日不修复建议止盈；中军看中国卫星竞价，前排锚点包括信维通讯等。主博 5/14 也提示商业航天有二番情绪，但反弹较多后不能追。

所以这条线仍可作为次选，但已经不是“看到航天就加分”。现在只看：

- 前排、中军、核心材料或设备股；
- 跌后能修复的底背离、缩量回踩、反包；
- 风险分不差、ATR 可控、止损位清楚的候选。

若候选只是商业航天概念，但 `MACD` 死叉、量价看空、趋势破位或高 ATR，就只能放观察。

## 5. 算力租赁 / 国产算力芯片：逻辑未灭，短线不追

价值锚仍认为算力租赁逻辑未变，利通电子、协创数据较强，字节相关润泽科技、华策等可观察；但主博 5/15 已经把多数算力定义为调整阶段。

当前处理方式：

- 算力租赁只保留强票和有支撑位的修复票。
- 国产算力芯片不追高，寒武纪、摩尔线程、海光、澜起科技、中国长城等只看回踩和消息修复。
- 光模块、存储、PCB 后面仍会反复炒，但买点要等调整，不追一致性高位。

## 6. 防守与轮动

电力仍是弱盘低吸和短线高手线，价值锚锚点包括大唐系、华电辽能等，不适合作为普通进攻主线。

金属和资源股更多是轮动：锂、锌、锗、黄金等会因为价格和避险情绪有弹性，但若 P1/P2 低、单日大跌或量价看空，不能接下跌惯性。

化工分化明显：电池相关氟化工优先，普通油气化工、新和成和万华这类阶段高位票先等回落。

创新药暂时仍偏防御，除非个股信号极强，否则不主动放进进攻主表。

## 7. 当前选股口径

- 第一优先级不是“题材名字”，而是 `主线贴合 + P1/P2 风险不过低 + P4 当日和五日连续性 + ATR 止损可接受 + 技术没有明显破位`。
- 半导体 + AI硬件可以入表，但只选回踩、抗跌和修复确认；不选高位放量追涨。
- 钠电、电池材料、氟化工若没有直接对应的优质候选，宁可空着，不用弱相关股票硬贴。
- 机器人方向权重上修，若分数结构干净，可以优先于普通主线外技术票。
- 航天、军工和商业航天只做修复，不做破位幻想。
- Phase7 允许交易只说明日期环境可做，不等于每只候选都能买；若 Phase7 变成 `no_trade`，当日只写观察池。

## 一句话新版结论

5/15 的核心变化是：`半导体 + AI硬件` 仍是中期主线，但短线不再追算力高位，光模块和存储也要等回踩；钠电、电池材料、氟化工和机器人边际增强；商业航天、算力租赁和国产算力只看修复和强票，破趋势就减仓。
