- `工业金属`：`4.28` 的最新长文把 `铜、铝、白银、锡、钨` 等又重新下调了一轮，除了锂矿和 `磷化铟` 这类情绪驱动的稀有金属外，多数都按阶段见顶后的震荡回落理解。

## 5. 当前更合适的交易节奏

综合后，当前更适合压缩成下面几条：

- 进攻主轴仍是 `AI硬件/算力`，但只做最强龙头和有业绩支撑的核心分支；多数泛算力不再追主升，更适合等 `15-30%` 回踩后的高位震荡机会。
- `光模块 / 存储芯片 / 少数核心PCB` 是 AI 主轴里的最高优先级，`国产算力链 / 半导体设备 / 国产替代` 只按延伸反弹和低吸理解。
- `新能源电池` 已进入更明确的二番战聚焦阶段，优先看 `钠离子 / 磷酸铁锂 / 中游绩优材料`，并更重视 `华阳股份 / 传艺科技 / 湖南裕能 / 多氟多` 这类辨识度更高的锚点。
- `电力 / 航天军工` 重新上修为更明确的防守承接方向，但只低吸趋势好的代表，不追题材扩散。
- `化工` 保留为稳但慢的周期修复线。
- `医药/创新药` 仍是防守修复方向，只在市场切防御时优先级明显上升。
- `消费电子` 只看龙头抱团，不升格主线。
- `商业航天 / 机器人 / 工业金属` 只按反抽和波段处理，不再作为默认进攻候补。
- `液冷`、弱势泛算力和AI营销噱头继续回避。
- 节前继续优先控制仓位，若先出现惯性下砸，反而更适合等风险释放后的安全买点。
