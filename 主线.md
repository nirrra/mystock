## 结论

按最新强调频率、是否被定义为五月核心、以及交易节奏是否清晰排序，当前主线更适合写成：

半导体 + AI硬件/算力（五月核心，`PCB/电子布/CCL/铜箔 + 光通信核心 + 存储芯片 + 先进封测前排 + 半导体靶材/材料设备 + 薄膜铌酸锂 + 海外链/国产算力轮动` 仍排第一；但存储芯片短线已经有阶段见顶提示，后续更适合等 `20-30%` 回踩再吃尾流） > 新能源电池/钠电/碳酸锂（最新主博观点明确“电池板块回来”“碳酸锂人弃我取”，从候补上修为次选反弹线，但仍以低吸和分歧确认优先） > 算力租赁（次选进攻线，有新的盈利模式预期差，但仍按低吸和强票处理） > 航天/商业航天/军工（次选趋势线，注意板块高切低，只聚焦核心个股） > 机器人（5/12 有异动，但板块合力稍差，目前更像指数带动的技术反弹） > 电力（只适合短线高手或弱盘防守低吸） > 化工（第一轮拉升基本完成，制冷剂、电池电解液和煤化工优于油气化工） > 稀土 / 锂镍钨铟等金属轮动观察 > 创新药（暂时待定，仍偏防御） > 工业金属高位品种 / 液冷 / 弱势泛算力继续回避追高。

落到每日选股时，主线只是第一层优先级，不替代风险过滤。`半导体 + AI硬件/算力` 内部也只保留 P1/P2 风险分不弱、趋势触发清楚、ATR 止损成本可接受的候选；如果主线票出现 P1/P2 任一明显偏低、顶背离、死叉或过高波动，就降级到“值得注意”，不用主线外纯技术票强行补位。

和上一版相比，最重要的变化是：

1. `存储芯片` 仍是半导体/AI硬件核心分支，但最新主博观点已经从“强势延续”变成“龙头接近压力、长鑫相关概念扩散、本周可能阶段见顶”；后面仍可维持高位震荡到中报甚至年底，但买点更适合等 `20-30%` 回踩，不适合一致性高潮追。
2. `半导体 + AI硬件/算力` 仍是五月第一梯队，其他博主 `2026.5.11` 继续把主线写成“半导体 + AI硬件”，并明确 AI硬件优先 PCB，光只看少数核心，先进封测前排可以继续看。
3. `AI硬件` 的细分新增/强化到薄膜铌酸锂，半导体靶材仍只看少数核心；同时最新节奏提示后续重点看量能，继续小心缩量。
4. `算力租赁` 仍是次选进攻线，利通、协创、东阳以及字节相关润泽是观察锚，但节奏仍偏低吸和强票确认。
5. `商业航天` 维持次选，但继续提示“板块高切低”，只能聚焦前排、中军和核心个股，不能按全面扩散追。
6. `新能源电池` 从候补反弹线上修为次选反弹线：5/12 主博明确电池板块回来，钠电继续看维科技术、华阳股份、传艺科技，碳酸锂继续看湖南裕能、天赐材料、多氟多等方向；但传艺减持跌停说明弹性线波动很大，仍要低吸和分歧确认。
7. `机器人` 5/12 有异动，中大力德短暂回归、三花智控较稳，但主博同时指出板块合力稍差，这一段更像指数带动的技术反弹，所以优先级仍低于 AI硬件/算力和电池反弹线。
8. `电力` 仍只适合短线高手；`化工` 仍是分支轮动：第一轮拉升基本完成，油气化工只适合大回踩波段，制冷剂、电池电解液和煤化工相对更好。

一句话概括：

当前 A 股的第一梯队仍收缩在 `半导体 + AI硬件/算力`，最新核心提示是 `PCB优先 + 光通信少数核心 + 存储芯片高位震荡 + 先进封测前排 + 半导体靶材/材料设备 + 薄膜铌酸锂 + 海外链/国产算力轮动`；存储芯片短线不宜追一致性扩散。5/12 的边际变化是电池、钠电、碳酸锂回到次选反弹线，算力租赁和航天仍是次选，机器人继续验证持续性，电力只适合短线高手。

## 1. 第一梯队主线

### 半导体 + AI硬件/算力：五月核心，进入海外链与国产算力轮动阶段

`其他博主.md` 在 `2026.5.11` 继续把主线写成“半导体 + AI硬件”，并明确“AI硬件优先 PCB，看文章自己搜对应龙头，光只看少数核心”。`2026.5.11` 还补充先进封测前排可以继续看、薄膜铌酸锂进入核心关注，同时提示后续量能和缩量风险。`2026.5.9` 又补充当前处于全球 AI 第二阶段，大概率围绕海外链和国产算力轮动，并提示先进封测核心企业仍低估。主博 `2026-05-06` 则把科技线拆成“业绩兑现分支”和“估值偏高分支”。所以这里的核心不是全面科技扩散，而是业绩确定、趋势未破、产业链位置清晰的窄主线。

内部优先级更适合拆成：

- `存储芯片`：主博最新新增观点继续确认存储强度，兆易创新、江波龙接近阶段压力，后面就算不继续新高也至少能维持高位震荡到中报，乐观能强到年底；但长鑫相关概念扩散到合肥城建、上峰水泥后，短线反而提示“本周阶段见顶”，后续更适合等 `20-30%` 回踩再参与尾流，不能追一致性高潮。
- `PCB / 电子布 / CCL / 铜箔`：AI硬件内部仍然 `PCB优先`，其他博主 `2026.5.11` 继续认为 PCB 拉长周期看更稳；电子布、CCL、铜箔继续找各自细分龙头。
- `光核心品种 / 光模块 / 光通信`：光方向只看持续新高、辨识度高、业绩进入兑现期的核心品种；最新仍只强调少数核心，说明光通信仍是 AI硬件内的重要分支，但不适合扩散到弱势泛光通信。
- `先进封测 / 半导体材料 / 靶材 / 设备 / 薄膜铌酸锂`：`2026.5.9` 单独提示先进封测核心企业仍低估，`2026.5.11` 继续写先进封测前排可以看，并点到半导体靶材、薄膜铌酸锂等少数核心品种，应把它们放进半导体优先分支。
- `国产算力链 / 国产算力芯片`：继续看寒武纪、海光、摩尔、中国长城、海光等大方向；若 CPU 美股大涨，中国长城、阿通、海光可以持有。但主博 `2026-05-06` 对寒武纪、摩尔线程、沐曦等仍不敢追涨，原因是估值偏高且互联网巨头自研投入会加剧硬件竞争。
- `算力租赁`：最新 `2026.5.11` 仍维持次选线，有新的盈利模式预期差，观察利通、协创、东阳，以及字节相关润泽；但仍要按低吸和强票确认处理，老位置要注意兑现压力。

这里的风险边界也很明确：

- 主博 `2026-05-06` 已经提醒，二季度可能是算力硬件全年的最好时刻，下半年如果出现放量拉升要格外小心，进入高位震荡一两个季度后动力会逐渐耗尽。
- 主博最新存储观点已经提示，存储从兆易创新、江波龙扩散到长鑫相关概念后，本周可能阶段见顶；这不改变中期高位震荡逻辑，但短线要降低追涨权重。
- 主博 `2026-04-29` 已经提醒，算力里除了光模块、存储芯片等技术含量高的方向，很多分支开始回吐，算力龙头也更适合等 `15-30%` 回踩后再参与高位震荡。
- `2026-04-24` 对新易盛跳水和 PCB 业绩兑现风险的提醒仍然有效：高位龙头只要环比增长松动，股价调整会很剧烈。
- `2026.5.11` 继续提示量能和缩量风险，所以半导体新主线不能理解成无条件追高，仍要等分歧、量能确认和低吸点。

当前更准确的表述是：

半导体 + AI硬件/算力是五月第一梯队，但它不是全面科技扩散，而是“业绩确定 + 产业链核心 + 持续趋势新高”的窄主线。选股时优先 PCB/电子布/CCL/铜箔、光通信核心、存储芯片、先进封测、半导体靶材/材料设备和少数算力核心；算力租赁可以作为次选进攻线观察，海外链和国产算力可以轮动看，但高开过猛要警惕预期兑现，国产算力芯片仍以低吸和支撑位反弹观察为主，弱势泛算力、液冷和纯概念营销继续回避。

### 新能源电池：5/12 回流，上修为次选反弹线

电池方向在主博 `2026-05-12` 里重新强化，原话核心是“电池板块今天回来”和“碳酸锂人弃我取”。这使电池从 `2026.5.11` 价值锚里的候补位置上修为次选反弹线，但它仍不是五月第一梯队核心，交易上更适合做分歧低吸、回踩承接和材料弹性确认。

最新口径包括：

- 主博 `2026-05-12` 写到电池板块回来，传艺科技因减持跌停，但对钠离子电池长期看好，继续关注维科技术、华阳股份，传艺科技也仍可观察，节奏是明显的波浪式往上。
- 主博 `2026-05-12` 单独强调碳酸锂“人弃我取”，重点关注湖南裕能，且认为天赐材料、多氟多估计也还有一波；碳酸锂涨价还会带动钠离子电池，维科技术和传艺科技仍在洗盘。
- 主博 `2026-05-08` 写到新能源电池继续，市场偏向小市值，传艺科技、永杉锂业、维科技术比较猛，而盐湖股份、赣锋锂业涨不动，近期传统锂矿巨头不宜配太多，钠电为主。
- 主博 `2026-05-07` 写到碳酸锂冲击 `20万`，盐湖股份和赣锋锂业还没到压力位，说明上游锂矿仍能给电池板块提供价格锚。
- 主博 `2026-05-07` 认为传艺科技涨停但板块共振一般，维科技术前面太猛、华阳股份稳但弹性不大，中短期还不如多氟多、天赐材料。
- 主博 `2026-05-06` 明确把钠离子电池定成年内重点：它不一定是业绩最好，但空间最大，维科技术、传艺科技、华阳股份继续作为观察锚。
- `2026.5.5` 其他博主写到：电池要“挑业绩好、持续趋势新高的”，核心看永杉、碳酸锂价格竞价表现，并继续点到材料、隔膜、天赐、湖南裕能、德方、维科、德业等方向。
- 主博 `2026-04-30` 明确说：磷酸铁锂还没到顶，但分化严重，最好遇到大回踩再来，重点还是湖南裕能。
- 主博 `2026-04-29` 仍确认电池二番战继续，但预期没有算力那么强烈；钠电要稳仍要看华阳股份、传艺科技、维科技术等节奏。
- 主博 `2026-04-28` 把电池拆成：上游锂矿稳住蓄势，中游湖南裕能、多氟多进入第二轮，钠离子比传统锂电下游更有弹性。

所以电池线当前的关键词是：

- `电池回流，二番战继续`
- `钠离子电池是空间线，短线偏小市值弹性`
- `磷酸铁锂未见顶但分化严重`
- `电解液 / 中游绩优材料`
- `碳酸锂 / 上游锂矿锚点`
- `户储`
- `不追一致性高潮，优先等分歧低吸和回踩确认`

当前更准确的定位是：

新能源电池从“候补反弹线”上修为“次选反弹线”。钠离子仍是空间弹性线，上游锂矿和碳酸锂是价格锚，电解液、中游材料和磷酸铁锂是业绩兑现线；但电池里很多弹性票波动很大，如果当日 `watchlist` 里的电池候选 P1/P2 明显偏低、顶背离、破 5 日/7 日趋势，仍不要因为题材回流而硬塞进主表。更优先的是主博点名方向里风险分能接受、回踩承接清楚、涨幅不过热的票。

### 航天 / 商业航天 / 军工：次选趋势线，注意高切低

航天方向在 `2026.5.7` 被其他博主写成次选核心线，`2026.5.10` 继续列为次选，并明确提示“注意板块高切低”。这说明航天已经不是完全回避线，但只能聚焦前排、中军和核心个股，不能把后排补涨当成主升。

但它的交易性质仍需要区分：

- `其他博主.md` 在 `2026-04-29` 已经写过“航天可以留着看下了，业绩披露得差不多了”，说明这条线从早前的纯题材风险里修复回来。
- `2026.5.7` 继续把航天列入次选和题材，说明它已经不能再按完全回避处理。
- 但主博此前对商业航天仍有“二番纯情绪”“反弹超过 30 个点不宜再追”的提醒，所以它不能升格为比半导体和 AI硬件更高的进攻主轴。

当前更合适的表述是：

航天/商业航天/军工已经上修为次选趋势线，选股时可以给主线权重，但只看前排、中军、趋势没坏和业绩确认度较好的票；若候选股仍在 `MACD` 下方、均线下方或被 TradingView 判成卖出，就只能放观察，不宜强行进主表。后排高切低要看承接，不能按题材扩散追。

## 2. 防守与轮动

### 电力：短线高手线，弱盘低吸才看

`其他博主.md` 在 `2026.5.7` 继续点到华电、大唐等电力方向，并明确写成“大盘差的时候可以考虑低吸”；`2026.5.10` 则把电力放到候补，并写明“只适合短线高手”。这说明电力的定位不是主动进攻，而是指数弱、仓位防守时的短线承接方向。

写入选股时，电力票只有在趋势保持、风险分低、止损清楚、ATR 不高时才适合入表；如果只是防守题材但技术破位，或波动成本已经过高，应降级为“值得注意”。

### 稀土：新增观察线，不直接升格主线

`2026.5.5` 至 `2026.5.6` 其他博主把稀土写成“可以关注”。这属于新增观察线，但最新 `2026.5.7` 没有继续强化，暂时还不足以把它升成第一梯队。

选股时可以把稀土、稀土永磁、先进金属材料里的强趋势票放到主线外补充或观察位；若同时出现底背离、量价看多、低风险和连续上榜，可以优先于普通主线外技术票。

### AI应用 / 机器人 / 汽车零部件：机器人有技术反弹，仍要看持续性

`其他博主.md` 在 `2026.5.7` 把 AI应用和机器人写成“机会”，主博 `2026-05-08` 则把机器人节奏进一步确认成“明显起势”。但 `2026.5.11` 的价值锚只保留“机器人依旧待观察持续性”，主博 `2026-05-12` 也明确说机器人板块异动但合力稍差，这一段更像指数带动的技术反弹。因此机器人可以保留观察，但不能直接替代 AI硬件、算力或电池反弹线。

机器人和汽车零部件仍可以看三花、拓普这类核心，中大力德、五洲、万向这类弹性要看盘中合力；靠近关键压力不追。AI应用没有被最新价值锚继续强化，只能在竞价、量价和持续性同时确认时看。选股时，相关候选若只是 Phase4 高分但不贴 AI硬件、算力租赁、电池或航天次选线，应优先放“值得注意”，不宜强行进主表。

### 化工：第一轮完成，分支明显分化

主博最新《2026年化工周期（2）》把化工线拆得更清楚：化工板块第一轮拉升基本完成，油气化工年内空间不是很大，大回踩可以做波段反弹，但不适合长线参与；制冷剂和电池电解液机会更大，煤化工更稳、周期性更明显。

当前更适合把化工写成分支轮动线，而不是统一的短线主攻方向。优先级更适合写成：`制冷剂 / 电池电解液 > 煤化工 > 普通油气化工波段`。

## 3. 继续回避或只按波段看的方向

- `工业金属`：碳酸锂、镍、钨、铟等有轮动弹性，但锡、镍靠近前高不能追，金银这一轮修复要赶紧跑；铜铝经过调整后更稳但上下空间不大。

## 4. 当前交易节奏

- 第一优先级是 `半导体 + AI硬件/算力`，但只做业绩确定、趋势新高、核心产业链和低吸位置，不追泛科技扩散，也不追预期利好兑现后的高开。
- 科技线内部优先看 `PCB/电子布/CCL/铜箔 + 光通信核心 + 存储芯片 + 先进封测前排 + 半导体靶材/材料设备 + 薄膜铌酸锂 + 海外链/国产算力轮动`；其中存储芯片短线已经有阶段见顶提示，只等回踩尾流，不再追长鑫相关概念扩散。
- `算力租赁` 已经单列为次选线，有盈利模式预期差，但只看强票和低吸，不在老位置追高。
- `航天/商业航天/军工` 是次选趋势线，核心个股优先，同时注意板块高切低；技术破位票不能因为题材在主线里就强行入表。
- `新能源电池` 上修为次选反弹线，钠电小市值弹性、碳酸锂价格锚和材料方向继续看；但传艺减持跌停说明弹性线波动大，仍要优先等分歧低吸、趋势承接和风险分确认。
- `机器人 / 汽车零部件` 有技术反弹，但板块合力稍差，只在个股 Phase、量价和盘中持续性明显更干净时入表；AI应用优先级进一步后移。
- `电力` 只适合短线高手或大盘弱时低吸，不追高。
- `稀土` 新增观察，优先看趋势强、低风险、有量价修复的票。
- `化工` 改成分支轮动线：制冷剂和电池电解液优先，煤化工偏稳，油气化工只看大回踩波段。
- `创新药` 暂时待定，只在防御环境或个股信号足够强时看。
- 当模型给出 `no_trade` 时，所有候选只作为观察池，不作为积极买入名单。
- 中期节奏上，二季度可能是算力硬件全年最好的阶段；下半年若出现放量拉升，应优先按高位震荡后段处理，而不是继续无条件加仓。短线上还要注意事件预期兑现风险。

## 一句话新版结论

五月核心仍是 `半导体 + AI硬件/算力`，2026.5.12 的最新边际变化是电池、钠电和碳酸锂回流，电池从候补上修为次选反弹线；但科技线仍优先 PCB、光通信少数核心、先进封测前排、半导体靶材和薄膜铌酸锂。存储芯片仍强但有阶段见顶提示，算力租赁与航天仍是次选，机器人只是技术反弹并等待持续性确认，电力只适合短线高手，弱势泛算力、液冷和国产算力芯片追高继续回避。
