## 人工精选核对清单（固定）

> 本段是每日选股前的固定人工核对项，后续新增每日选股记录时不要覆盖、删除或移动。新的日期段落从本段后面开始，按交易日新到旧排列。本轮采用新选股标准，旧日期记录已清空，`2026.5.15` 为第一条交易日记录。

当前盘后只使用 `P1 / P2 / P4 / 板块P9 / 长期主线指数 / 短期主线指数 / 龙头指数`。`P3 / P5 / P7 / P8 / P10` 已废弃，不展示，也不作为推荐依据。

固定核对：

- 主线先行：先看 `watchlist_sectors`、`sector_mainline_daily_tracking.xlsx` 和 `主线.md`，再看个股。
- 长期主线：长期主线指数高，说明板块长期被资金反复炒作，符合“三三制”反复轮动特征。
- 短期主线：短期主线指数、当日涨幅、上涨家数占比和龙头表现共同判断最近 5-20 个交易日谁在带动市场。
- P9：只用于板块/概念买入分，不替代主线判断；P9 高但当日弱或博主未确认时，要写清楚。
- 个股：只从确认主线或可买板块中挑，优先 `P1/P2/P4` 均衡、`P4五日均分` 高且 `std` 不高、龙头辨识度高或 pattern 结构清楚的股票。
- Pattern：只提供结构，不是独立买点。模式命中但 P1/P2 低时，必须降低仓位预期并写清止损风险。
- 涨幅过滤：当日涨幅 `> 9.9%` 的涨停或近涨停票不进入最终买入表。
- 仓位：按 `D = 2 * ATR14`，建议总仓位 `min(40%, 0.02 * P / (0.85 * D))`。买入分四批：开仓点、回踩结构线或回撤约 `D/2`、上涨到 `1R` 后回踩、创新高。
- 风控：初始止损为 `P - D`；涨到 `1R` 后止损上移到成本；之后按最高点 `- 2.5 * 实时ATR` 动态止损。买入后 5 个交易日仍不能脱离成本区或站稳结构位，执行时间止损。
