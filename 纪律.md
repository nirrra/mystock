# 交易纪律

本文件只记录已经准备买入或已经持有的股票。任何一笔交易在买入前，必须先写清持股理由、买入价、总仓位、R 值、止损和第一次止盈位置。

## 固定公式

```text
D = R = 2 * ATR14
初始止损价 = P - D
单笔总最大亏损 = 2% * 账户总资金
有效平均止损距离 = (1 - 0.3) * D + 0.3 * 0.5D = 0.85D
理论总仓位比例 = 0.02 * P / (0.85D)
最终总仓位比例 = min(40%, 理论总仓位比例)
```

如果已知 `ATR%`，且 `ATR%` 用百分数数值表示，例如 `4` 表示 `4%`：

```text
最终总仓位% = min(40, 117.65 / ATR%)
```

止损移动：

```text
开始止损 = P - R
第三次加仓前，最大止损保持 P - R
价格到 P + R 并完成第三次加仓确认后，止损上移到 P
继续上涨后，动态止损 = 持仓后的最高价 - 2.5 * 实时 ATR14
动态止损只上移，不下移
```
