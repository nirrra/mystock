# 项目详细说明

本文档记录当前主线。系统目标已经从“直接按个股分数选股”调整为“先确认市场主线，再从主线板块中选个股”。旧的 Phase3、Phase5、Phase7、Phase8、Phase10 已从用户入口、watchlist 和选股文档中移除。

## 当前主线

`daily-screening` 是盘后主流程：

```text
update
-> update-sector-membership
-> macd
-> atr
-> P1 tail risk
-> P2 barrier risk
-> P4 Alpha158 return
-> pattern
-> sector leaders
-> P9 sector buy score
-> watchlist_stocks
-> intraday_pool Top200
-> watchlist_sectors
```

`intraday-screening` 是盘中主流程：

```text
读取前一交易日 intraday_pool / 最近 watchlist_sectors
-> 更新候选股盘中行情
-> 用盘中临时 K 线重算 P1/P2/P4
-> 根据板块龙头盘中涨幅生成板块强度
-> 输出盘中候选表
```

## 核心指标

| 指标 | 含义 | 当前用途 |
| --- | --- | --- |
| P1 | 全市场尾部风险模型，分数越高风险质量越好。 | 个股风险过滤和综合排序。 |
| P2 | Triple-barrier 交易型风险模型，分数越高交易风险越低。 | 个股风险过滤和综合排序。 |
| P4 | Qlib Alpha158 + LightGBM 收益回归模型，分数越高上涨质量越好。 | 个股收益排序主轴。 |
| P9 | 板块未来 20 日收盘上涨超过 5% 的概率模型。 | 评价板块是否有低位布局价值。 |
| 长期主线指数 | 板块过去较长周期内是否持续受到资金关注。 | 找长期主线板块。 |
| 短期主线指数 | 板块最近一段时间是否正在带动市场。 | 找当下活跃主线。 |
| 龙头指数 | 个股在所属板块过去两年里的领涨能力。 | 从强板块里挑龙头股。 |

## 主线 + 个股选股口径

人工选股先判断主线，再看个股。主线判断需要把量化结果和 `主线.md` 中的博主观点结合起来：

1. 长期主线：用长期主线指数找长期受资金关注、可反复炒作的方向。这类板块更符合“三三制”里可长期跟踪、反复轮动的主线。
2. 近期主线：在长期主线里，用短期主线指数、最近涨幅、上涨家数和 P9 找最近带动市场、疑似处于主升初期或可开始低位分批建仓的方向。
3. 当日强度：盘后看板块当日涨幅和龙头表现；盘中看 `intraday_sector_strength` 里龙头股盘中平均涨幅。判断主线是增强、延续、减弱，还是发生切换。

如果量化结果和 `主线.md` 的博主观点一致，优先级最高；如果量化强但博主观点未确认，先观察；如果博主提到但量化强度不足，也先观察。若长期主线、短期主线和 P9 头部强度都弱，按市场主线混沌处理，减少新开仓。

## 候选池口径

盘后 `watchlist_stocks` 合并两类来源：

1. Pattern 命中股：不做 P1/P2 风险门槛，只排除当日涨停股。
2. P1/P2/P4 综合排序 Top20：排除 P1/P2 最高风险 20%，并按 centered risk 综合分排序。

盘后 `watchlist_sectors` 使用排序名额制，不再使用固定阈值。先按 `长期主线指数` 取 Top100 作为主池；再在长期主线 Top100 内部，按 `短期主线指数` 标记 Top10，按 `P9买入分` 标记 Top10。短期主线和 P9 高分不单独拉入非长期主线板块。如果 P9 模型不存在，P9 留空，但长期主线和短期主线仍正常输出。

主线判断不能只看入选数量，还要看强度分数。若 Top100 里头部长期主线指数、短期主线指数和 P9 分数都不强，应判断为市场主线偏混沌，减少新开仓，只观察少数高分共振方向。

最终人工选股时，优先考虑：

1. 股票所属板块属于长期主线。
2. 该板块最近开始走强、P9 较高，或盘中/盘后强度明显增强。
3. 个股龙头指数高，或在 `watchlist_stocks` 中 P1/P2/P4 综合表现较好。
4. Pattern、MACD、ATR 只作为交易辅助信息，不替代主线判断。

## 交易纪律摘要

仓位按 ATR 反推。设总资金为 `E`，初始买入价为 `P`，`D = 2 * ATR14`，总最大亏损为 `2% * E`。当前分批方案在第三次加仓前的平均止损距离约为 `0.85D`，理论总仓位比例为：

```text
理论仓位比例 = 0.02 * P / (0.85 * D)
             = 0.01 / 0.85 * P / ATR14
```

实际仓位仍受单一标的最大 40% 限制。

分批与出场：

1. 开仓点买第一批。
2. 回踩结构线或突破确认买第二批。
3. 上涨到 1R 后的回踩买第三批。
4. 创新高买第四批。
5. 初始止损为 `P - R`。
6. 上涨到 1R 后止损抬到 `P`。
7. 1.5R 止盈 20%-30%，3R 再止盈 20%-30%，剩余仓位用实时 `最高点 - 2.5 * ATR` 动态止损。
8. 加入时间止损：若买入后若干交易日内没有按预期走强，应减仓或退出。
