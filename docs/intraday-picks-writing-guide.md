# 选股-日中.md 写入指导

本文用于把盘中结果整理到 [选股-日中.md](/C:/Users/wdyab/Desktop/wdy/stocks/选股-日中.md)。如果文档中已经有对应日期的选股列表，必须先删除该日期已有内容并完全忽略旧内容，只根据最新盘中筛选结果、最新板块池、[主线.md](/C:/Users/wdyab/Desktop/wdy/stocks/主线.md) 和本指导重新列出。

当前盘中只使用 P1、P2、P4、最近的 `watchlist_sectors`、`watchlist_sector_leader_pool`、`intraday_sector_strength`、板块 P9 和盘中涨幅。盘中不处理 B 形态池，不展示 P3、P5、P7、P8、P10。

## 输入

优先读取：

- `reports/intraday_screening/intraday_sector_strength_YYYY-MM-DD.csv`
- `reports/intraday_screening/intraday_watchlist_a_YYYY-MM-DD.csv`
- `reports/intraday_screening/intraday_watchlist_a1_recent_mainline_YYYY-MM-DD.csv`
- `reports/intraday_screening/intraday_watchlist_a2_rotation_expected_YYYY-MM-DD.csv`
- 最近一个交易日的 `reports/watchlists/watchlist_sectors_YYYY-MM-DD.csv`
- 普通模式来源池：前一交易日 `reports/watchlists/watchlist_sector_leader_pool_YYYY-MM-DD.csv`
- 若当天跑过全市场盘中刷新：当天 `reports/watchlists/watchlist_sector_leader_pool_YYYY-MM-DD.csv`
- `主线.md`

盘中结果只使用临时日 K，不写入正式日线。不要重新识别 Pattern，也不要补写跟踪股。

## 写作顺序

每个日期只保留一个三级标题，不按 10:00、11:40、14:30 等时段拆分。当天后续盘中更新时，覆盖该日期整段内容，不追加新的时间段小节。

每个日期按这个顺序写：

1. 标题：`### YYYY.M.D`。
2. 盘中市场状态。
3. 盘中主线判断。
4. 关注主线表。
5. A1 近期强势。
6. A2 轮转预期。
7. 总结：逐股日中判断。

## 盘中市场状态与主线判断

盘中先判主线，再看个股。必须把量化结果和 `主线.md` 的博主观点结合：

开头必须先用列项判断盘中市场状态，不要把几句话挤在一段里：

```md
盘中市场状态：

- 状态：强主线 / 多主线 / 主线衰退 / 主线轮转 / 混沌。
- 依据：长期主线、近期主线和龙头盘中均涨幅是否集中。
- 操作：可低吸 / 只看主线 / 等回踩 / 暂缓新开仓。
```

随后用列项写主线分析：

- **长期主线**：先看最近 `watchlist_sectors` 中长期主线指数高的方向，判断哪些属于长期受资金追捧、可反复炒作的板块。
- **近期主线**：看短期主线指数、P9 标签和最近几日涨幅，判断最近一段时间谁在带动市场。
- **当日强度**：看 `intraday_sector_strength` 中龙头股盘中平均涨幅，判断主线当日是增强、延续、减弱，还是出现切换。
- **主线轮转**：判断盘中强势方向是否仍是原主线，还是资金切到新的板块。
- **博主观点对照**：说明 `主线.md` 与盘中量化结果是否一致。

示例：

```md
盘中主线：

- 机器人仍是长期主线，且盘中龙头强度增强。
- 商业航天延续，但力度弱于昨日。
- 钠电属于回调观察，不追。
```

## 关注主线表

在盘中主线判断后、日中选股表前，列出当前最值得关注的主线板块和概念。

#### 关注主线

| 板块/概念 | 长期主线 | 短期主线 | P9买入 | 龙头盘中均涨幅 | 龙头股 | 主线状态 | 操作判断 |
|---|---:|---:|---:|---:|---|---|---|

`主线状态` 只用短词：增强、延续、减弱、切换、回调企稳、混沌。`操作判断` 只用短词：可低吸、等回踩、只观察、不追。

`龙头股` 必须带盘中涨幅，保留一位小数，用中文括号，例如：`德明利（0.0%）/生益科技（0.0%）/深南电路（0.0%）`。盘中表格不用显示板块盘中涨幅，只显示龙头盘中均涨幅。

## 日中选股

盘中只输出 A1/A2：

- A1 近期强势：盘中候选池只要求短期强度 Top10 板块、真实属于该关切板块、盘中涨幅不过热、P1>20、P2>20。最终选股时，优先挑 P9 仍高、个股混合分高、龙头指数高、盘中涨幅不过热的股票。
- A2 轮转预期：盘中候选池只要求 P9 Top10 板块、真实属于该关切板块、盘中涨幅不过热、P1>40、P2>40。最终选股时，优先挑混合分高、龙头指数高、P4 不弱、低位可分批建仓的股票。
