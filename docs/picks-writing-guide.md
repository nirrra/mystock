## 市场状态与主线判断

先判断主线，再看个股。主线判断必须结合量化结果和 `主线.md` 的博主观点：

开头必须先用列项判断市场状态，不要把几句话挤在一段里：

```md
市场状态：

- 状态：强主线 / 多主线 / 主线衰退 / 主线轮转 / 混沌。
- 依据：长期主线、短期主线、P9 和当日龙头表现是否集中。
- 操作：可低吸 / 只做主线 / 控仓观察 / 暂缓新开仓。
```

随后用列项写主线分析：

- **长期资金追捧主线**：看长期主线指数，找长期反复被资金炒作、符合“三三制”反复轮动特征的板块或概念。
- **最近带动市场主线**：看短期主线指数、最近涨幅、上涨家数、板块涨幅，判断最近 5-20 个交易日谁在带动市场。
- **潜在轮转主线**：看 P9 买入分，找仍有 20 日上涨预期的方向。
- **当日强度变化**：结合板块涨幅、龙头表现和 `主线.md`，判断主线是持续、增强、减弱，还是发生切换。
- **博主观点对照**：说明 `主线.md` 与量化结果是否一致。

如果量化主线和博主观点一致，写成优先方向；如果不一致，写清楚“量化强但主线.md 未确认”或“博主提到但量化强度不足”。若头部长期主线、短期主线和 P9 分数都不强，按市场主线混沌处理，减少新开仓。

## 三条路线

A1 近期强势：从 `watchlist_a1_recent_mainline` 选。它代表长期主线 Top100 内短期强度 Top20 的广义候选，只在程序层面要求真实属于该关切板块并通过公共过滤 `涨幅<=9.9%、P1>20、P2>20`。最终选股时，再人工优先选择板块 P9 仍高、个股混合分高、龙头指数高、涨幅不过热的股票。

A2 轮转预期：从 `watchlist_a2_rotation_expected` 选。它代表长期主线 Top100 内 P9 Top20 的潜在轮转方向，程序层面要求真实属于该关切板块、通过公共过滤，且 `P1>40、P2>40`。最终选股时，再人工优先选择混合分高、龙头指数高、P4 不弱、处于低位可分批建仓的股票。

B 形态符合：从 `watchlist_b_pattern` 选。它只代表 Pattern 结构候选，必须通过公共过滤 `涨幅<=9.9%、P1>20、P2>20`。B 不与 A1/A2 混排。若 B 同时属于 A1/A2 主线或潜在轮转主线，可以提高关注级别。
